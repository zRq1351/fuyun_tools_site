Windows AI 划词 + 截图录屏 OCR + 剪贴板管理

把剪贴板、截图录屏 OCR 和 AI 划词，做成一条真正可用的效率链路

fuyun_tools 常驻系统托盘，通过统一快捷键管理文字与图片历史；在 Windows 上进一步打通选词翻译、快捷截图录屏与图片 OCR 识别的完整路径。

核心能力

  • 文本剪贴板自动记录、去重、搜索、分类、回填
  • 图片剪贴板缩略图管理、双击回填、全屏预览、磁盘限额
  • Windows 划词工具栏支持翻译 / 解释 / 复制 / 自定义提示词 / 网页搜索
  • Windows 截图标注工具：框选/圆形/箭头/文字/画笔/马赛克/取色器，支持长截图与固定到屏幕
  • Windows 录屏胶囊支持开始/暂停/恢复/停止，系统音频+麦克风采集，按键说话
  • Windows 固定图片窗口右键 OCR 识别，双引擎离线运行
  • Windows 应用启动器（Alt+Q）：扫描开始菜单，模糊搜索快速启动
  • Windows 文档管理器（Ctrl+Shift+D）：索引与仓库双模式，FTS5 全文搜索
  • 支持 DeepSeek、通义千问、小米 Mimo 与自定义 OpenAI 兼容服务
  • 数据备份与恢复：设置/历史/图片一键导出，自动备份与回滚
  • 全局热键、托盘菜单、开机自启、应用更新、系统诊断

功能模块

剪贴板管理

自动去重记录文本历史，支持分类、固定、容量保护，键盘方向键与鼠标双路径交互回填。

图片剪贴板

自动捕获图片历史，异步缩略图加载，磁盘配额管理，支持分类、标记与全屏预览。

AI 划词助手

Windows 下监听文本选择与鼠标事件，自动弹出工具栏，支持翻译/解释/复制/自定义提示词与网页搜索，流式输出。

截图标注

区域截图与自动窗口检测，提供框选/圆形/箭头/文字/画笔/马赛克/取色器等标注工具，支持撤销重做、长截图与固定到屏幕。

图片 OCR

固定图片窗口右键触发 OCR 识别，双引擎（Windows 原生 + PaddleOCR）离线运行，结果在独立文本窗口展示。

录屏与音频采集

录屏胶囊支持开始/暂停/恢复/停止，支持系统音频与麦克风设备选择，按键说话，可配置帧率与码率参数，ffmpeg 按需下载。

应用启动器

扫描开始菜单快捷方式，支持模糊搜索、分类管理、自定义命令，键盘导航快速启动应用（Alt+Q）。

文档管理器

索引与仓库双模式，支持 PDF/Word 内容提取与 FTS5 全文搜索，分类管理、拖拽导入与导入撤销（Ctrl+Shift+D）。

AI 服务配置

内置 DeepSeek、通义千问、小米 Mimo 提供商，支持自定义 OpenAI 兼容服务，API Key 本地加密保存，提供连接测试。

备份与诊断

设置/历史/图片一键导出与恢复，支持自动备份与回滚保护；内置系统诊断面板监控存储、依赖与性能状态。

系统集成

应用常驻托盘，支持 6 组全局热键，具备开机自启、自动更新、主题切换（暗色/亮色/护眼），零数据上传，OCR 完全离线。

软件使用流程

01

基础配置与启动

启动后常驻托盘，建议首先进入设置配置 AI 服务与 API Key、调整各类历史上限与快捷键设置。

02

文本与图片剪贴板

使用 Ctrl+Shift+Z / Ctrl+Shift+X 快速唤出文字/图片窗口，通过方向键、回车或双击回填到当前焦点应用。

03

AI 划词助手

Windows 任意应用中选中文本自动弹出工具栏，支持一键翻译、解释、复制及自定义提示词流式结果展示。

04

截图标注与 OCR

使用 Ctrl+Shift+S 区域截图，通过标注工具编辑后复制/保存/固定到屏幕，在固定窗口右键触发 OCR 文字识别。

05

录屏与音频采集

进入设置启用录屏后通过录屏胶囊开始录制（Alt+R）；首次自动检查并按需下载 ffmpeg，支持按键说话控制麦克风。

06

应用启动器

使用 Alt+Q 唤出启动器，输入关键词模糊匹配应用名称，回车快速启动；支持分类管理与自定义命令。

07

文档管理器

使用 Ctrl+Shift+D 打开文档管理器，拖拽导入文件，按分类整理，FTS5 全文搜索快速定位，支持导入撤销。

08

数据备份与恢复

在设置中手动导出备份包，或开启自动备份按日/周/月定期执行；支持合并/覆写两种恢复策略与回滚保护。

实现架构

前端层

多页面 Vue 结构分别承载剪贴板、图片管理、截图编辑、录屏胶囊、启动器、文档管理、设置、划词工具栏与结果窗口。

Tauri 运行层

Rust 侧统一管理全局快捷键、托盘菜单、窗口行为、插件能力与状态同步。

业务服务层

拆分为 clipboard / image_clipboard / ai / screenshot / recording / launcher / document / backup / diagnostic 服务，涵盖历史监听、流式 AI、截图标注、录屏采集与数据备份。

Windows 扩展特性

通过全局鼠标/键盘 Hook 与文本选择引擎，驱动划词工具栏与截图取色器等交互，结合 WASAPI/WGC 实现系统级音频与录屏采集。
