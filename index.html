Windows AI 划词 + 跨平台剪贴板管理

把剪贴板和 AI 划词，做成一条真正可用的效率链路

fuyun_tools 常驻系统托盘，通过统一快捷键管理文字与图片历史；在 Windows 上进一步打通选词、翻译、解释、复制的完整路径。

技术栈：Vue 3 + Element Plus + Tauri 2 + Rust AI SDK：async-openai（OpenAI 兼容） 当前最新版本：加载中...

核心能力

  • 文本剪贴板自动记录、搜索、分类、回填
  • 图片剪贴板缩略图管理、双击回填、全屏预览
  • Windows 划词工具栏支持翻译 / 解释 / 复制
  • 支持 DeepSeek、通义千问、小米 Mimo 与自定义 OpenAI 兼容服务
  • 全局热键、托盘菜单、开机自启、应用更新

功能模块

剪贴板管理

支持历史上限策略、分类管理、键盘与鼠标双路径交互，适合高频复制粘贴场景。

图片剪贴板

自动识别并持久化图片历史，支持搜索、分类、双击回填与大图预览。

AI 划词助手

Windows 下监听文本选择与鼠标事件，自动弹出工具栏并流式输出翻译或解释结果。

AI 服务配置

内置多提供商并支持自定义服务，API Key 本地加密保存，提供连接测试能力。

系统集成

应用常驻托盘，支持热键唤起文本/图片窗口，具备自动更新与开机自启。

数据与安全

历史记录与配置本地存储，生产环境默认不落日志文件，降低敏感信息暴露风险。

软件使用流程

01

启动与唤出

启动后常驻托盘，使用 Ctrl+Shift+Z 快速打开剪贴板窗口。

02

快速回填

通过方向键、回车或双击，选择历史项并回填到当前焦点应用。

03

AI 划词处理

Windows 选中文本后自动弹出工具栏，一键翻译、解释与复制。

04

配置模型与策略

在设置中配置 AI Provider 与模型，同时调整历史上限和分类策略。

实现架构

前端层

多页面 Vue 结构分别承载剪贴板、图片管理、设置、划词工具栏与结果窗口。

Tauri 运行层

Rust 侧统一管理全局快捷键、托盘菜单、窗口行为、插件能力与状态同步。

业务服务层

拆分为 clipboard/image_clipboard/ai 服务，包含历史监听、流式 AI 调用与错误处理。

Windows 扩展特性

通过鼠标监听与文本选择模块，驱动划词链路及工具栏的即时显示与交互。
